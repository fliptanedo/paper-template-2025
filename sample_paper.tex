\section{Introduction}

This is a template for my papers and lecture notes. The sample file includes \LaTeX{} snippets for common components of such documents. It also includes some (opinionated) suggestions for best practices when preparing a \LaTeX{} document.

There are now three templates:
\begin{enumerate}
    \item \verb!sample_paper.tex!: (this document) for papers.
    \item \verb!sample_shortnote.tex!: for quick notes; no author list, no bibliography, no hassle.
    \item \verb!sample_lecturenotes.tex!: a hefty framework for writing extended lecture notes that can be printed and used as a textbook. Features: chapter/section, large margins for comments, BibLaTeX citations as footnotes, chapter appendices. 
    \item \verb!sample_nicedocument.tex!: a professional document that uses local fonts (XeLaTeX), colors, and a nice footer. Useful for job applications.
\end{enumerate}

There is example text located in \texttt{./examples/}. These include common chunks of code and commentary on best practices.
